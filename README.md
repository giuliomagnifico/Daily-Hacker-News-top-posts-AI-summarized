## Usage

Import the JSON into n8n and adjust it as you prefer (e.g. 10 or 50 articles, or a different prompt to chatGPT). 

To change the number of articles, modify the number in the HTTP Request node: `https://hn.algolia.com/api/v1/search?tags=front_page&hitsPerPage=20`
